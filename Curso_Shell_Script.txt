File Globbing x REGEX 

File Globbing - É uma forma que o bash tem para selecionar arquivos e diretórios na interface (ls,cp, mv etc..)

# Uso do * 

	ls -l Aula*	# Lista todos os arquivos com o nome Aula e qualquer caracter que venha DEPOIS

	-rw-r--r-- 1 ubuntu ubuntu 0 Sep  1  2017 Aula1
	-rw-r--r-- 1 ubuntu ubuntu 0 Sep  1  2017 Aula10
	-rw-r--r-- 1 ubuntu ubuntu 0 Sep  1  2017 Aula11


	ls -l *11	# Lista todos os arquivos e diretório que termine com 11

# Uso do ? 	
	
	ls -l Aula?	# Lista todos arquivos e diretórios que tenha somente UM caracter após a letra a

	-rw-r--r-- 1 ubuntu ubuntu 0 Sep  1  2017 Aula1
	-rw-r--r-- 1 ubuntu ubuntu 0 Sep  1  2017 Aula2
	-rw-r--r-- 1 ubuntu ubuntu 0 Sep  1  2017 Aula3
	

	ls -l Aula??	# Lista todos arquivos e diretórios que tenha DOIS caracter após a letra a

	-rw-r--r-- 1 ubuntu ubuntu 0 Sep  1  2017 Aula10
	-rw-r--r-- 1 ubuntu ubuntu 0 Sep  1  2017 Aula11
	-rw-r--r-- 1 ubuntu ubuntu 0 Sep  1  2017 Aula12

# Uso da colchetes LISTA 
    
	ls -l Aula[123]	# Lista todos os arquivos e diretórios que TERMINAM em 1,2 e 3 

	-rw-r--r-- 1 ubuntu ubuntu 0 Sep  1  2017 Aula1
	-rw-r--r-- 1 ubuntu ubuntu 0 Sep  1  2017 Aula2
	-rw-r--r-- 1 ubuntu ubuntu 0 Sep  1  2017 Aula3


    ls -l Aula[1-5] # Lista todos os arquivos e diretórios que terminam do 1 ao 5 

	-rw-r--r-- 1 ubuntu ubuntu 0 Sep  1  2017 Aula1
	-rw-r--r-- 1 ubuntu ubuntu 0 Sep  1  2017 Aula2
	-rw-r--r-- 1 ubuntu ubuntu 0 Sep  1  2017 Aula3
	-rw-r--r-- 1 ubuntu ubuntu 0 Sep  1  2017 Aula4
	-rw-r--r-- 1 ubuntu ubuntu 0 Sep  1  2017 Aula5


	

# Uso das chaves STRINGS 

	ls -l {Aula,AULA}11	# Lista os arquivos que tenha o mesmo valor na string passada dentro das chaves 

	-rw-r--r-- 1 ubuntu ubuntu 0 Sep  1  2017 Aula11
	-rw-r--r-- 1 ubuntu ubuntu 0 Sep  1  2017 AULA11



REGEX - Função de selecionar um padrão de texto (grep, egrep e fgrep)

egrep - Expressões regulares extendidas  (Deixa o comando mais pesado)
grep  - Expressões regulares simples
fgrep - Não usa expressões 


# Usando egrep com REGEX 

	egrep "[Ll]inux" texto.txt					# Imprime todas as palavras com o nome Linux iniciando com L maiúsculo ou l minúsculo 
	
	egrep "b[aei]g"	 texto.txt 					# Imprime todas as palavras qe começam com b e que tenham as ocorrências a,e,i na palavra e 
	que terminem com g

	egrep "^Linux"	 texto.txt    				# Imprime todas as ocorrências que começem como a palavra Linux
	egrep "^#" texto.txt 						# Imprime todas as ocorrências que começem a linha com # 
	egrep -v "^#" texto.txt						# Ele não mostra as ocorrências que começem com o #

	egrep "Linux$" texto.txt				    # Sempre que a ocorrência for no final da linha será mostrada 

	egrep "^$" texto.txt						# Será mostrado todas as ocorrências de linhas vazias 
	egrep -v "^$" texto.texto					# Não será mostrado as ocorrências de linhas que estão vazias

--> Caracters especiais 


# A letra g pode existir nenhuma vez ou várias vezes

	egrep "b[a-i]g*" texto.txt 		


# Letra g deve existir pelo  									menos uma vez ou diversas vezes

	egrep "b[a-i]g+" texto.txt 		

# A letra g pode não existir, mas que apareça somente uma vez. 

	egrep "b[a-i]g?" text.txt 		

# Imprime a linhas que tenha um caracter entre a letra O e o nome Linux

	egrep "O.Linux"	text.txt 		
	
# Exige que o caracter anterior exista, e que possa existir qualquer coisa entre O e Linux. 


	egrep "O.*Linux" texto.txt 	

#
	egrep "[Ll]inux." texto.txt
	egrep "[Ll]inux\." texto.txt 

# Deleta os comentários da saída 

	sed '/^#/d' text.txt

# Substitui a palavra Linux e começando com l ou com L pela palavra Unix de forma global, ou seja será alterado todas as ocorrências encontradas. 

	sed 's/^[Ll]inux/Unix/g' texto.txt
	  	


--> Resumo 

	* NENHUMA ou varias vezes ocorrências 
	+ UMA ou varias vezes 
	? NENHUMA ou apenas UMA VEZ
	. Pode ser qualquer caracter entre duas letras  
	.* Exige que o caracter anterior exista e que posso existir qualquer caracter entre as duas letras
